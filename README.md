# newtab-poem

## description
Расширение для браузера, которое на новой открытой вкладке показывает случайное стихотворение.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
